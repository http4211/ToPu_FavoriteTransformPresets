# ToPu_FavoriteTransformPresets
お気に入りのトランスフォーム設定を最大12まで保存＆ワンクリックで適用

<img width="111" height="28" alt="image" src="https://github.com/user-attachments/assets/e44bda7a-391c-4ce6-a46a-024d5427ab85" />

・12までツールヘッダーに表示できるようになっています。

・アイコンはスロットごとに自由に指定できます。

・UIを表示する位置は四か所から指定できます。

・設定したすべてのプリセットのエクスポート/インポートに対応しています。

---
### スロット詳細設定



現在のトランスフォーム設定を右上のスロットの<img width="30" height="28" alt="image" src="https://github.com/user-attachments/assets/586d3f77-0aa8-4e30-8cbc-5c9605827cf2" />uiから取得できます。

[アイコン名：]はアイコンを指定できます。https://extensions.blender.org/add-ons/icon-viewer/?utm_source=blender-4.4.3 などのアドオンから指定のアイコン名を検索してカスタマイズしてください。

適応する際影響を受けたくない項目は、適応のチェックを外せば現在の設定から影響を受けません。
スナップ設定は個別で影響を受けたくない箇所を右側にあるチェックで調整できます。
